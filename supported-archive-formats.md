# Supported Archive Formats

| Format Name | File Extension   | Description           |
| ----------- | ---------------- | --------------------- |
| 7z          | .7z              |                       |
| zip         | .zip             | ZIP file              |
| bztar       | .tar.bz2 / .tbz2 | bzip2'ed tar-file     |
| gztar       | .tar.gz / .tgz   | gzip'ed tar-file      |
| tar         | .tar             | uncompressed tar file |
| xztar       | .tar.xz / .txz   | xz'ed tar-file        |
